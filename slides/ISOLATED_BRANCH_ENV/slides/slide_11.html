📋 前提条件

必要なもの

  • Kubernetesクラスタ
    自前のクラスタ or GKE/EKS/AKS
  • Cloudflareアカウント
    ドメイン管理・API Token発行
  • Google Cloud Platformアカウント
    Container Registry (GCR)の使用
  • GitHubリポジトリ
    GitHub Actionsの使用

必要な環境変数（GitHubシークレット）

変数名 説明
ARGOCD_TOKEN ArgoCD APIトークン
GCP_SA_KEY GCPサービスアカウントキー
KUBECONFIG_CONTENT Kubernetesクラスタの接続情報
